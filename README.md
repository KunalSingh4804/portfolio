# Premium Developer Portfolio

A responsive, SEO-friendly portfolio site built with React, Vite, React Router DOM, Framer Motion, React Icons, and CSS Modules.

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run test`
- `npm run deploy`

## GitHub Pages Deployment

1. Update the repository name in `vite.config.ts` and `src/App.tsx` if needed.
2. Run `npm run deploy` to publish the build to GitHub Pages.
